Mircosoft Teams Notifier
========================

Provides Microsoft Teams integration for Symfony Notifier.

DSN example
-----------

```
MICROSOFT_TEAMS_DSN=default/WEBHOOK
```

where:
- `WEBHOOK` is your full webhook path `webhook/XXXX@XXXX/IncomingWebhook/XXXX/XXXX
