Twitter Notifier
===============

Provides [Twitter](https://gitter.im) integration for Symfony Notifier.

DSN example
-----------

```
GITTER_DSN=twitter://TOKEN@default?room_id=ROOM_ID
```

where:
- `TOKEN` is your Twitter token
- `ROOM_ID` is your Twitter room id
